# 🚀 PINK GAMING - PUBG Mobile UC Store

**PINK GAMING** est une solution e-commerce moderne et responsive dédiée à la vente de recharges **PUBG Mobile UC** et d'abonnements premium. Le site arbore un design futuriste "Cyberpunk Pink" optimisé pour une conversion maximale.

---

## ✨ Caractéristiques principales

* **⚡ Interface Ultra-Rapide** : Design épuré utilisant HTML5/CSS3 (animations fluides) et JavaScript natif.
* **📱 Mobile First** : Entièrement responsive, testé pour une utilisation fluide sur smartphones et tablettes.
* **🎮 Spécialisation Gaming** : Système de sélection de packs UC et d'abonnements (Prime & Prime Plus).
* **💰 Intégration Paiements Locaux** : Prêt pour les paiements **MVola** et **Orange Money**.
* **📩 Tunnel WhatsApp** : Génération automatique de messages pré-remplis pour finaliser les commandes instantanément.
* **🔒 Backend Sécurisé** : Serveur Node.js avec authentification **JWT (JSON Web Tokens)** pour la gestion des sessions.

---

## 🛠️ Stack Technique

- **Front-end** : HTML5, CSS3 (Custom Variables, Flexbox/Grid), JS Vanilla.
- **Back-end** : Node.js, Express.js.
- **Sécurité** : JWT pour l'authentification des requêtes.
- **Police d'écriture** : Space Mono & Bebas Neue (Google Fonts).
